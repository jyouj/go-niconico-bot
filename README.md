# go-niconico-bot
単語を入力するとニコニコ動画でその単語の一番人気の動画を返してくれるGolang製Discord Bot

## Install
```
$ go get github.com/jyouj/go-niconico-bot
```

## Setup
### Use direnv
`DISCORD_CLIENT_ID`を`.envrc`に書き込んでください。

### Add niconico bot to your server
Botをサーバーに参加させるために認証してください。
